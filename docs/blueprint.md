# **App Name**: LockerAdmin Pro

## Core Features:

- Locker Management: View and manage individual lockers, including status (available, occupied, maintenance) and details.
- User Account & Locker Assignment: Manage user accounts (create, edit, delete) and assign lockers to users.
- Payment History & Status: View payment history and status (paid, pending, overdue) for lockers. Display a payment forecast for the upcoming 30 day period.
- System Activity Logs: Access a tool which parses system activity logs. Identify anomalies and summarize events that may need admin attention. Format events from most important to least important based on context.
- Role-Based Dashboards: Role-based dashboards for Admin, Super Distributor, Distributor, and Retailer roles.
- Account Management: Account creation and password visibility for lower-level users, with role-based restrictions.
- Activation/Removal Reports: Generate daily, weekly, and monthly reports on account activations and removals.
- Device Tracking: Track device code generation, transfers, and returns.
- User/Device Lists: List users and devices with filtering, counts, and detailed information.
- Support Info Management: Update contact and support information displayed to users.
- Banner Upload: Upload banners and graphics to Cloud Storage for use in the mobile app.

## Style Guidelines:

- Background: Dark charcoal grey (#333333) for a professional and secure feel.
- Primary color: Muted gold (#B8860B) for subtle accents and highlights, conveying elegance.
- Accent color: Dark gold (#8B4513) for interactive elements and important notifications.
- Font: 'Inter', a sans-serif font, for clear readability across the admin panel.
- Use minimalist, line-based icons in muted gold to represent locker status, user actions, and payment information.
- Maintain a clean, structured layout with clear sectioning for locker management, user accounts, payment history, and activity logs.
- Implement subtle animations for loading states and transitions to enhance user experience without being distracting.
